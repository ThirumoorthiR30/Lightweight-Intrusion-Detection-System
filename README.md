# Hybrid Deep Learning-Based Intrusion Detection System

## Overview

This project presents a Hybrid Deep Learning-Based Intrusion Detection System (IDS) designed to detect and classify malicious network activities using advanced Artificial Intelligence and Deep Learning techniques. The framework integrates Genetic Algorithm (GA) based feature selection, Convolutional Neural Networks (CNN), and Bidirectional Gated Recurrent Units (Bi-GRU) to improve cyber threat detection performance.

## Key Features

* Genetic Algorithm (GA) for optimal feature selection
* Deep Learning-based intrusion detection
* 1D Convolutional Neural Network (CNN) for spatial feature extraction
* Bidirectional GRU (Bi-GRU) for temporal dependency learning
* Binary and multiclass attack classification
* End-to-end machine learning pipeline
* Network traffic analysis and cyber threat detection
